# Mini Bank RESTful SpringBoot Application
- Authentication (JWT)
- Accaunt managment
- Financial transactions
- Notifications (Kafka)
- Database migrations (Liquibase)
- SpringDoc OpenAPI (Swagger)

# Stack
- Java 17
- Spring Boot 2.7.18
- Spring Security
- Hibernate
- PostgreSQL
- Liquibase
- Apache Kafka
- Docker & Docker Compose
- Maven
- JUnit 5 & Mockito
- Swagger
